# NBA 王朝经营 · Dynasty GM 2026

基于 **2026年5月真实 NBA 赛况** 的总经理模拟游戏。

## 背景新闻（已内置）

- 尼克斯横扫骑士，自 1999 年以来首次打进总决赛（布伦森、唐斯）
- 西部决赛：雷霆 3-2 领先马刺（SGA 32 分 G5；文班亚马 G4 33 分）
- 2026 总决赛 6 月 3 日开打
- 2025-26 薪资帽 $154.647M、第二围裙 $207.824M

## 玩法

1. 选择 30 支球队之一担任 GM
2. 管理阵容、合同、交易、自由市场、选秀
3. 运营球馆收入、球迷与媒体关系
4. 聘请教练组，制定争冠/重建策略
5. 模拟常规赛 82 场 + 完整季后赛 bracket + **赛程表**（2-2-1-1-1 主场、日期、转播、七场四胜）
6. **王朝存档**：5 个本地槽位 + JSON 导入导出
7. **化学反应**：球星双人组默契、团建提升、影响战力
8. **经纪人事件**：续约、交易申请、商业活动等随机决策
9. **中文解说直播**：逐节比分 + 解说词条回放单场

## 运行

直接用浏览器打开 `index.html` 即可，无需安装依赖。

```bash
# Windows
start index.html

# 或本地服务器
npx serve .
```

## 文件结构

- `index.html` — 主界面
- `data.js` — 球队、球员、CBA 规则、化学反应、经纪人
- `engine.js` — 比赛模拟、季后赛 bracket、财务、交易
- `broadcast.js` — 中文解说单场生成
- `save.js` — 王朝模式 localStorage 存档
- `ui.js` — 界面渲染
- `styles.css` — 样式

## 推荐体验

选择 **尼克斯 (NYK)**、**雷霆 (OKC)** 或 **马刺 (SAS)** 可体验与真实 2026 季后赛一致的明星阵容。
